# CodeLibs
Repositório com algumas bibliotecas PHP adaptadas por mim para o [CodeIgniter](https://codeigniter.com/) 2 ou superior.

### Algumas libs adaptadas

* [OpenBoleto](https://github.com/kriansa/openboleto)

### Como utilizar?

Para você que já tem os conhecimentos prévios de como funciona o CodeIgniter, cada biblioteca tem sua respectiva pasta com os
arquivos necessários dentro de `application\third-party`, e o loader em `application\libraries`.
